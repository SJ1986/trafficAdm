기본 가이드

신규 페이지 생성시 DOCTYPE은 HTML5이며, 파일 인코딩은 UTF-8로 생성한다.
페이지 생성 및 유지보수 진행시 하단 코드를 참고하여 진행한다.
추가되는 css는 ux-admin.css 하단에 추가되는 해당 매뉴의 css 를 위치한다. 

                                    <!DOCTYPE html>
                                    <html lang="ko">

                                    <head>
                                        <meta charset="UTF-8">
                                        <meta http-equiv="X-UA-Compatible" content="IE=edge">
                                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                        <link rel="stylesheet" href="../resource/css/ux-admin.css">
                                        <script type="text/javascript" src="../resource/js/jquery-3.6.0.min.js"></script>
                                        <script type="text/javascript" src="../node_modules/simplebar/dist/simplebar.js"></script>
                                        <script type="text/javascript" src="../resource/js/common.js"></script>
                                        <script type="text/javascript" src="../resource/js/main.js"></script>
                                        <title>자율주행 데이터 공유센터</title>
                                    </head>
                                    <body>
                                        <div class="wrap">
                                            <!-- header -->
                                            <header class="header">
                                                <div class="header__inner"></div>
                                            </header>
                                            <!-- container -->
                                            <section class="container">
                                                <div class="container__inner">
                                                    <div class="content">
                                                        <div class="content__inner">
                                                        </div>
                                                    </div>
                                                </div>
                                            </section>
                                            <!-- footer -->
                                            <footer></footer>
                                        </div>
                                    </body>
                                    </html>

CSS 정의

css 추가는 각 페이지 별로 따로 만들어 연결하되 공통 스타일은 common.css 쪽, 고정 style은 fix.css에 포함한다.
페이지 별 css는 sub__페이지명.css로 만들되 페이지별 head쪽 ux-admin.css 아래에 선언한다.
plugIn css의 경우 css폴더 내 lib 폴더에 넣어서 연결한다. 

                                    <link rel="stylesheet" href="../resource/css/ux-admin.css">
                                    <link rel="stylesheet" href="../resource/css/sub__manage.css">
